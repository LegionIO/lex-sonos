# lex-sonos

Sonos speaker control for [LegionIO](https://github.com/LegionIO/LegionIO). Control Sonos devices from within task chains.

## Installation

```bash
gem install lex-sonos
```

## Requirements

- Ruby >= 3.4
- [LegionIO](https://github.com/LegionIO/LegionIO) framework
- Sonos speakers on local network
